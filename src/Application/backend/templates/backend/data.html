Общая информация о разделе "Data"


Раздел «Data» содержат подразделы, которые позволяют через web-интерфейс приложения поддерживать следующий функционал:

  1. Совершать CRUD-операции (CReate/Update/Delete) по всем таблицам;
  2. После любого вида модификации таблиц пользователю выводиться измененная таблица;
  3. Поддерживается импорт и экспорт данных для каждой таблицы из файлов/в файлы с расширением .csv

Интерфейс данного раздела разбит на две части:

  1. Левая сторона содержит список таблиц и CRUD-операций предназначена для выполнения;
  2. Правая сторона выводит необходимый результат/форму в соответсвии с выполненной операцией.
