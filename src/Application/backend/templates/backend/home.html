Добро Пожаловать на проект Web Retail Analitycs


В данном проекте разработано web-приложение на языке программирования Python (3.11) для проекта SQL3 Retail Analitycs, c использованием паттерна MVC и фреймворка Django (4.2).

Приложение поддерживает осуществление CRUD-операций, импорт/экспорт таблиц, вычисление всех персональных предложений и их экспорт в файл разрешения .csv через web-интерфейс приложения.

Дополнительно реализован функционал логирования действий пользователя и deploy web-приложения на nginx.

Краткая навигация по web-приложению:

  1. На вкладе "Login/Logout" Необходимо проити авторизацию на проекте, не авторизированный пользователь не имеет доступа к остальным разделам проекта;
  2. На вкладе "About" Вы познакомитесь с разработчиком трудившегося над данным проектом;
  3. На вкладе "Data" у Вас будет возможность познакомиться с текущей базовой данных (БД), а также совершить CRUD - операции;
  4. На вкладе "Proposals" реализован функционал вычисления всех персональных предложений
