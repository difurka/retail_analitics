Общая информация о разделе "Proposals"


Раздел "Proposals" содержат подразделы ("Manual" и "Proposals"), которые позволяют через web-интерфейс приложения поддерживать следующий функционал:

  1. Ознакомиться с общей информацией о каждой функции персональных предложений, с указанием описания входных и выходных данных;
  2. Вычисления всех функций персональных предложений и их экспорт в файл разрешения .csv;
  3. В случае необходимости введения параметров для выполнения функции, графический интерфейс предоставляет форму для ввода данных;
  4. Если введенные аргументы/SQL-запрос были некорректны, приложение выдавает ошибку о некорректности введенных данных и предлагать повторную попытку ввод;

Интерфейс данного раздела разбит на две части:

  1. Левая сторона предназначена для выполнения выбора необходимой функции;
  2. Правая сторона выводит результат в соответсвии с выполненной операцией.
